Shopware is a modern, open-source e-commerce framework used by thousands of merchants worldwide. Whether you're building a custom storefront, integrating with external systems via its REST APIs, or extending functionality through plugins and apps, Shopware gives you everything you need.

## Key concepts

<Steps>
  <Step title="Install Shopware">
    Use the Symfony Flex template or Docker to get a local instance running. See the [installation guide](/installation).
  </Step>
  <Step title="Understand the architecture">
    Learn about the Data Abstraction Layer (DAL), the three-API model, and the event system before building. Start with the [architecture overview](/architecture/overview).
  </Step>
  <Step title="Choose your extension method">
    Use **Apps** for cloud-compatible, lightweight extensions via webhooks and scripts. Use **Plugins** when you need direct access to the Shopware process and database.
  </Step>
  <Step title="Build and test">
    Create entities, subscribe to events, add API routes, and build admin UI modules. Use `bin/console` for scaffolding and migration management.
  </Step>
</Steps>
